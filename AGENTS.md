# AI Agent 协作规范

## 沟通语言声明
- 沟通语言声明：所有文档、提交信息、评审讨论及与 AI Agent 的交互，统一使用简体中文。

## 任务协同
- 明确需求、约束与交付物路径后再进入实现，必要时用简要计划同步。
- 任何涉及命令行执行、代码修改或生成文件的操作，需在回复中说明目的与结果。

## 产出要求
- 文档类输出使用 Markdown，确保结构清晰、语义准确。
- 代码修改应遵循项目既有风格与编码规范，并附带必要的上下文描述。
- 若存在无法完成或需确认的事项，应在结论前列出等待决策的点。

## 版本管理
- 禁止主动回滚或覆盖用户已有的未提交变更，除非用户明确指示。
- 提交前建议运行必要的检测/测试，并在说明中记录执行情况。
