# AI Agent 协作规范（入口）

本文件为根入口，智能体必须先阅读本页，再按需查看 `agents/` 下的分拆规范。若目标路径存在更近的 `AGENTS.md`，以“就近”为准；若需覆盖规范，请在 `agents/session-summary.md` 记录原因。

## 分拆文件索引（必须遵守）
- `agents/rules-process.md`：沟通语言、基本协作流程、智能体工作区与程序化检查。
- `agents/rules-git.md`：提交/分支/推送约束、版本与自检要求、SemVer/日期/CHANGELOG 对齐。
- `agents/rules-docs.md`：文档写作与链接规则、Index/SRS/SDD 模板、用例/锚点规范。

## 使用说明
- 先读本入口，再阅读相关分拆文件；执行任务时以“最近的 AGENTS + 本索引”共同约束行为。
- 若新增子目录 `AGENTS.md`，需声明其覆盖范围，并在 `agents/session-summary.md` 备注差异。
- 显式的人类指令优先生效；若与规范冲突，优先执行指令并记录。
