# 外部入口安全与信任 SRS（占位）

> 占位文件：列出需覆盖的事实要点，后续按需补充。

## 范围与基线
- 适用于 URI Handler / 外部深链 / 外部登录等入口的安全、身份与信任约束。
- 基线与 `uri-and-links-srs.md` 一致。

## 需覆盖的事实要点
- 参数白名单与长度/类型校验；禁止直接拼接为 Shell/文件路径。
- 破坏性操作（删除/覆盖/推送）必须要求用户确认。
- Token/签名/一次性凭证的验证要求；过期/重放的处理。
- 隐私/敏感信息不得出现在可共享 URI；日志脱敏。
